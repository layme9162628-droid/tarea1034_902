Resumen

Un sistema diseñado para la automatización industrial y el control autónomo

Se priorizó la sincronización precisa entre el software de procesamiento de imágenes y la respuesta física del hardware, asegurando un flujo de trabajo continuo y eficiente en la clasificación de objetos.

Este proyecto abordó la necesidad de automatizar la selección de materiales en líneas de producción mediante el uso de lógica embebida. La propuesta se concentró en el desarrollo de una grúa transportadora con control angular y lineal, sincronizada con una banda transportadora y una cámara web como sensor principal.

El objetivo fue suprimir la intervención manual en tareas de clasificación repetitivas, optimizando la precisión del posicionamiento mecatrónico y la robustez del software ante variaciones del entorno real.

Resultados

Impacto directo en la automatización del proceso

La entrega se diseñó para operar de forma totalmente autónoma, implementando algoritmos computacionales de detección cromática que coordinan simultáneamente motores paso a paso y servomotores para la clasificación precisa de cubos rojos, amarillos y verdes.

Resultado

Eliminación total del error humano en la etapa de separación de objetos, logrando un sistema funcional validado con posicionamientos repetibles tanto en la grúa como en el avance de la cinta transportadora.

Aprendizaje

Migración hacia el espacio de color HSV (en lugar de RGB) para dar robustez al software ante cambios de iluminación ambiental, necesidad de mitigación de ruido eléctrico en entradas de reloj de integrados síncronos y optimización mecatrónica usando encoders para retroalimentación en tiempo real.
