Grúa controlada por microcontrolador capaz de clasificar cubos de colores

Este proyecto consiste en un sistema automatizado de clasificación y transporte de objetos mediante una grúa controlada por un microcontrolador. El objetivo principal es identificar cubos de colores (rojo, amarillo y verde) colocados sobre una cinta transportadora, para luego trasladarlos hacia contenedores específicos según su color. La detección se realiza mediante una cámara web, que envía la información al microcontrolador, el cual coordina los movimientos de la grúa y la cinta para completar el proceso de manera autónoma.
